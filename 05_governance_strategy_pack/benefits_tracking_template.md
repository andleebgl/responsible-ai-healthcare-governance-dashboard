## Benefit Types

| Type | Description |
|------|-------------|
| Efficiency | Reduces time or effort for staff |
| Clinical Outcome | Improves patient health outcomes |
| Patient Experience | Improves the patient journey or satisfaction |
| Operational | Improves how the organisation runs day to day |
| Financial | Reduces costs or increases revenue |

## How to Use This Template

1. When a use case is approved, record the expected benefit and set a baseline metric
2. Set a realistic target metric based on what the use case should achieve
3. After the pilot or implementation, record the current metric
4. Update the status to In Progress, Achieved, Partially Achieved, or Not Achieved
5. Review quarterly at the governance board to track progress

## Review
This register should be updated after each pilot phase and reviewed quarterly alongside the risk register and prioritisation matrix.
